## 团号规则（主键约定）

格式：`YX-YYMM-XX-NN`

- `YX` —— 研学固定前缀
- `YYMM` —— 出发年月（两位年 + 两位月）
- `XX` —— 目的地英文代号（BJ/SH/AH/HZ/XA…）
- `NN` —— 当月同目的地序号（01 起）

示例：`YX-2408-AH-01`（24 年 8 月 安徽 第 1 团）

**约束**：手工填写，全局唯一。建议在 Base 视图里加唯一性筛选辅助查重。
**联动**：飞书群名建议带团号后缀；行程文档命名也带团号便于检索。

## 字段说明

字段按业务段分组，描述见 Base 内字段 `description`。

### 主表「团组」分组

| 段 | 字段 |
|---|---|
| 基本信息 | 团号 / 团组类型 / 出发日期 / 返回日期 / 天数 / 成人数 / 学生数 / 年龄段 / 出发地 / 目的地 |
| 销售/咨询 | 主题/目标 / 预算 / 需领队/总控 / 含沙龙安排 / 其他销售要求 |
| 商务 | 报价 / 成本 / 地接成本 / 单房差 / **利润**(公式) / **毛利率**(公式) |
| 客方 | 组团方名称 / 学校名称 / 客方对接人 / 客方联系方式 / 合同号 / 实收款 |
| 行程 | 最终行程文档(URL) / 交通方式 / 住宿要求 / 用餐要求 |
| 人力 | 导游 / 领队 / 生活老师 / 队医/摄像 |
| 流程状态 | 状态 / 主控团控 |
| 关键时间节点 | 教委上报截止 / 行前会日期 / 出团日 / 尾款日 |
| 系统 | 创建时间 / 最后更新 / 关联6张子表 |

### 状态字段取值

`咨询中 → 已立项 → 行前准备 → 行中 → 行后 → 已结算 → 已归档`
