# 研学团组管理系统 · 使用指南

> 解决两个核心痛点：**视图拥挤** + **录入笨重**

---

## 痛点 1：看板拥挤怎么办

不要再用单一"状态看板"看所有团。**按时间切片 + 异常聚焦** 是飞书最佳实践。

### 新视图地图（按场景挑视图）

| 场景 | 用这个视图 | 视图类型 |
|---|---|---|
| 🔥 **每天上班第一眼** | `亮红灯` | grid（仅显示有问题的团） |
| 📅 **这周要操盘哪些团** | `这周出团` | grid（距出团 0-7 天） |
| 📋 **本人责任田** | `我负责的` | grid（按主控团控筛选） |
| 🗓️ **下周做哪些行前准备** | `下周出团` | grid（距出团 8-14 天） |
| 📊 **当月业务一览** | `本月剩余` + `状态看板` | grid + kanban（≤60 天） |
| 🎯 **季度计划** | `下月预备` + `远期(2月外)` | grid |
| 📍 **按目的地协调资源** | `按目的地分组` | grid（group_by 目的地） |
| 👶 **按客户群体看** | `按类型分组` | grid（group_by 团组类型） |
| 📜 **历史团回顾** | `已归档` | grid（已结算 + 已归档） |
| 🗓️ **排期密度** | `甘特(团组排期)` | gantt（按状态着色） |
| 📈 **近 3 月**日历视角 | `行前关键日历` | calendar |

### 关键技巧

- **公式字段「距出团(天)」** 是切片基石：今天起 ±N 天自动归类
- **公式字段「红灯原因」** 自动标记：合同未签 / 教委逾期 / 尾款逾期
- 「状态看板」已加 `距出团≤60` 筛选，远期团不再挤占看板

## 痛点 2：录入怎么提效

不要让任何角色对着 grid 横拉填字段。**每个角色一个表单**，只露相关字段。

### 3 个表单 + 1 个仪表盘

| 表单 | 用户 | 字段数 | 入口 |
|---|---|---|---|
| 📋 立项快速登记 | 销售/咨询 | 15 个（精简） | 见下方 URL |
| ✏️ 行前信息补全 | 团控 | 主表非基本字段 | 见下方 URL |
| 👥 学员名单收集表 | 班主任/家长 | 18 个，13 必填 | 见下方 URL |
| 📊 团组运营驾驶舱 | 管理层 | 12 个组件 | 见下方 URL |

### 表单工作流（推荐）

```
[销售拿到客户意向]
   ↓ 立项快速登记 (3 分钟)
[团组进入"已立项"状态]
   ↓ 团控接手
[T-14 行前信息补全]（一次性补齐 报价/成本/人力/客方/...）
   ↓ T-10 行前会前
[向学校发学员名单收集表链接]
   ↓ 班主任逐条提交
[T-7 检查必备项齐备] (走"亮红灯"视图)
   ↓
[T-3 出 5单1书]
   ↓ 出团
```

### 启用表单分享（你需要做一次的事）

**API 暂不支持自动启用对外分享**，需要 **手工** 操作一次：

1. 打开 Base → 切到对应表单视图
2. 右上角点「分享」→ 选「任何人可填写」
3. 复制公开链接，回填到本指南的 URL 表里

启用后链接对外永久有效，可贴到群、邮件、微信。

## 仪表盘组件清单

「团组运营驾驶舱」12 个组件：

| 组件 | 类型 | 说明 |
|---|---|---|
| 标题 | text | 入口说明 |
| 本月团数 | statistics | 出团日在本月的团数 |
| 本月营收(应收) | statistics | 报价合计 |
| 本月毛利 | statistics | 利润字段合计 |
| 待结款总额 | statistics | 应收且未完成的金额 |
| 团组状态分布 | pie | 全量团按状态切片 |
| 近期团数(按月) | column | 月度趋势 |
| 团组类型占比 | ring | 学生/亲子/师生/企业 |
| 地接社合作排行 | bar | 历史合作次数排序 |
| 反馈类型分布 | pie | 客人意见/投诉/签单等 |
| 收付款方向汇总 | column | 应收/实收/应付/实付 |
| 目的地热度 | wordCloud | 哪些目的地团多 |
