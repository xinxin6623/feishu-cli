# workflows/create-tour

立项一个新研学团组，**一条命令**完成：

1. 校验团号格式 + 唯一性（查 Base 主表）
2. 建飞书内部团控群
3. 在「研学团组管理」下建团组子文件夹（含 4 个业务子目录）
4. 在 Base 主表「团组」写入立项行（状态自动置为"已立项"）

## 用法

```bash
./run.sh <团号> <类型> <目的地> <出发日期 YYYY-MM-DD> <天数> [主题]

# 预览不真跑
./run.sh YX-2607-HZ-01 学生研学 杭州 2026-07-10 6 "宋韵研学" --dry-run

# 真跑
./run.sh YX-2607-HZ-01 学生研学 杭州 2026-07-10 6 "宋韵研学"
```

## 团号规则

`YX-YYMM-XX-NN`

- `YX`：研学固定前缀
- `YYMM`：出发年月（24/25 年 = 24/25，两位）
- `XX`：目的地 2-3 位英文代号（BJ / SH / AH / HZ / XA …）
- `NN`：当月同目的地序号（01 起）

校验在脚本前置完成，团号已存在会直接拒绝。

## 依赖

- lark-cli ≥ 1.0.39，已 `auth login`
- jq（macOS: `brew install jq`）

## 退出码

| code | 含义 |
|---|---|
| 0 | 成功 |
| 2 | 参数错误 / 团号格式错误 |
| 3 | 团号已存在 |
| 4 | 建群失败 |
| 5 | 建文档目录失败 |
| 6 | Base 写入失败 |
